# FAQ

## Is KeepClean safe to try?

Yes, but the safest first test is always `Disable Keyboard` because the built-in trackpad stays active.

## Does it disable external keyboards or mice?

No. This version only works with the built-in keyboard and built-in trackpad.

## Does it stay open in the background?

No. Closing the main window fully quits the app.

## Does it connect to the internet?

Not on its own. The app does not include analytics, sync, or background network activity. The only browser actions are the links you click in the About tab.

## What happens if macOS denies permission?

KeepClean shows a plain-language error and leaves your Mac usable.

## Why is there a helper process?

The helper is only for the strict timed keyboard + trackpad clean. It helps the app recover automatically when the timer ends, even if the main window is gone.

## Which mode should I use most often?

Use `Disable Keyboard` for quick wipes and routine cleaning. Use the timed keyboard + trackpad clean when you want a stricter no-input cleaning pass.

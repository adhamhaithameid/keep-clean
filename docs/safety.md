# Safety Notes

KeepClean is deliberately conservative.

## Safer Option

`Disable Keyboard` is the safest mode.

- It disables only the built-in keyboard.
- The built-in trackpad stays active.
- You can immediately turn the keyboard back on from the same button.

## Strict Timed Option

`Disable Keyboard + Trackpad` is for real cleaning passes where you do not want accidental clicks either.

- It uses the timer you choose in Settings.
- It auto-recovers when the timer ends.
- It does not try to manage external keyboards or mice in this version.

## Why the App Quits on Window Close

KeepClean fully quits when you close the window so it does not sit around in the background and surprise you later.

## Recommended Habit

Use the keyboard-only mode first any time you are testing a new Mac, new macOS version, or a fresh install.
